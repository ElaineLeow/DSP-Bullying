# EduSafe: Predicting Bullying Risk Among Students

## Overview

EduSafe is a machine learning project that predicts bullying risk among students using behavioral and emotional survey data.

## Objectives

- Identify students at risk of bullying
- Support early intervention
- Provide classroom-level risk analysis

## Technologies Used

- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit

## Machine Learning Models

- Logistic Regression
- Random Forest
- XGBoost

## Features

- Bullying risk prediction
- Individual student assessment
- Classroom-level analysis
- Automated reporting and visualization
